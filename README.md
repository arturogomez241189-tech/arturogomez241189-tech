*   **[HackTheBox - Cap Writeup](/arturogomez241189-tech/HTB-Cap-Writeup):** Análisis y pwn completo de la máquina "Cap". *   **[Nmap Scanning Practice](/arturogomez241189-tech/scan-with-nmap-practice):** Auditoría y descubrimiento de activos/servicios de red mediante técnicas avanzadas de escaneo con Nmap.
